# locust压力测试

### 运行环境
```
python3
locust
```
### python3安装
```
$ wget https://www.python.org/ftp/python/3.7.4/Python-3.7.4.tgzc
$ tar -xzvf Python-3.7.4.tgz
$ cd Python-3.7.4
$ ./configure 
$ make && make install
```
### locust 安装 
```
$ python3 -m pip install locustio
```

### 执行压力测试
```
$ python3 run.py
```
